# color2048
这是安卓版的2048

![界面](https://github.com/shenjingfs/MarkDown/blob/master/images-folder/Color2048.png)

## 玩法
* 每次可以选择上下左右其中一个方向去滑动
* 所有的数字方块都会往滑动的方向靠拢，相同数字的方块在靠拢时会相加
* 系统也会在空白的地方随机出现一个数字方块
* 合成2048这个数字就算成功

## 特性
* 流畅的动画
